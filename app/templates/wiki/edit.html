Markdown Guide

Text Formatting

  • **bold** → bold
  • *italic* → italic
  • `code` → inline code
  • ~~strikethrough~~

Headings

  • # Heading 1
  • ## Heading 2
  • ### Heading 3

Lists

  • - Item for bullet points
  • 1. Item for numbered lists

Links & Images

  • [text](url) → link
  • ![alt](image-url) → image

Other

  • > Quote → blockquote
  • ```python\ncode\n``` → code block
